La Liste de Course

Fiche E5
rapport
Voir

L'objectif de ce projet était de créer une application mobile afin d'aider à faire une liste de courses.

Cette application devait être écrite en Java Android et utilisera ORMLite pour gérer une base de données SQLite qui contiendrait les données.


L' utilisateur devait pouvoir configurer l'application avec :

- création / modification / suppression d'un produit (libellé, quantité).
- création / modification / suppression d'une recette (ensemble de produits).

Une fois les produits et les recettes crées, les possibilités suivantes devaient être disponibles :

- création / modification / suppression d'une liste de courses.
- ajout d'un produit à une liste de courses.
- ajout de produits à une liste en sélectionnant une recette.

Après la liste de courses établie, lors des courses en elles-mêmes, l'utilisateur devait pouvoir :

- cocher un élément d'une liste (pour signaler qu'il est ajouté au caddie).

Outils devant être utilisés :


- Utilisation du logiciel Git pour travailler en équipe.
- ORMLite.
- Android Studio.

Le site est composé de deux partie :

- Une application Java Android.
- Une base de données SQLite.

Matériel Utiliser :

- Un pc sous windows.
- Téléphone fonctionnant sous Android pour tester l'application
Ou téléphone émulé.

Logiciels et librairies :

- Android Studio.
- Git.
- ORMLite.
